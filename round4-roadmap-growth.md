# 顺时 ShunShi 迭代路线图

## 10 年可扩展路线

```
Phase 1: MVP (0-12个月)
├── 核心对话功能
├── 节气养生内容
├── 基础习惯追踪
├── 免费版本
└── 目标: 10万用户

Phase 2: 家庭与付费 (12-24个月)
├── 家庭系统
├── 订阅体系
├── 记忆系统
├── AI Router 优化
└── 目标: 100万用户

Phase 3: 生态雏形 (24-36个月)
├── 健康硬件接入
├── 企业版
├── Skills 平台
└── 目标: 500万用户

Phase 4: 养老陪伴 (3-5年)
├── 老年模式
├── 语音陪伴
├── 养老机构合作
└── 目标: 1000万用户

Phase 5: 全球化 (5-10年)
├── 海外版本
├── 本地化
├── 健康服务连接
└── 目标: 1亿用户
```
